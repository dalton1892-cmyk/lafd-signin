# LAFD Digital Sign-In Sheet

A standalone web app for LAFD program teams to capture participant sign-ins with digital signatures, configurable dropdowns, and easy email/print to `grants@lacity.org`.

## Features
- Configurable dropdown menus (Program, Location, Instructor, Session, Date)
- Add unlimited custom extra fields (text or dropdown)
- Digital signature + initials pad (works with stylus, finger, or mouse)
- Upload your team's logo (PNG/JPG)
- Email sign-in records to `grants@lacity.org` via Gmail
- Print-friendly layout with signatures
- CSV export
- QR code sharing (when hosted at a public URL)
- All data persists in browser localStorage â€” no server required

## Usage
Open `index.html` in any web browser. No installation or internet connection required.

## Sharing / QR Code
Host these files on [GitHub Pages](https://pages.github.com/) (free) to get a shareable URL and QR code. Set GitHub Pages source to the root of the main branch.
